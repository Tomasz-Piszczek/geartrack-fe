# GearTrack Frontend

A React-based web application for tracking and managing industrial equipment, tools, and employee assignments.

## Features

- **Equipment Management**: Track machines and tools with assignments
- **Employee Management**: Manage employee profiles and equipment assignments  
- **Authentication**: Secure login with Google OAuth
- **Dashboard**: Overview of equipment and employee data
- **Responsive Design**: Works on desktop and mobile

## Tech Stack

- React 19 + TypeScript
- Tailwind CSS
- TanStack Query
- React Router DOM
- Google Auth Library
- Vite

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Development server
- `npm run build` - Production build
- `npm run lint` - Run linter
